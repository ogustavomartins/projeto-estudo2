Cores em HTML e CSS

A emoção das cores

Jamais devemos subestimar o poder das cores em um design, elas podem influenciar que o usuário fique horas navegando, porém as mesmas podem fazer o usuário sair em menos de 5 minutos. Segundo especialistas, as cores passam emoção para o subconciente das pessoas, mesmo que na maioria dos casos isso não seja feito de forma conciente.

Veja:

  • Vermelho: Paixão, amor, energia, urgência, perigo, raiva.
  • Amarelo: Felicidade, otimismo, calor, alegria, mas também alerta e ansiedade.
  • Laranja: Entusiasmo, criatividade, calor, amigável, alegria.
  • Rosa: Afeto, romance, suavidade, inocência, doçura (tons pastéis).
  • Azul: Confiança, segurança, calma, profissionalismo, mas também tristeza e vazio.
  • Verde: Natureza, saúde, crescimento, frescor, tranquilidade, equilíbrio.
  • Roxo/Púrpura: Mistério, sofisticação, espiritualidade, criatividade, luxo.
  • Preto: Elegância, poder, mistério, formalidade, luto (ocidente)
  • Branco: Pureza, simplicidade, limpeza, modernidade, paz.
  • Marrom: Terra, estabilidade, conforto, aconchego, mas também melancolia.

Círculo Cromático

O círculo cromático é uma representação visual e organizada das cores em um formato circular.

Ele é formado pelas três cores primárias: amarelo, vermelho e azul. A partir das cores primárias, surgem as cores secundárias: laranja (amarelo + vermelho), violeta (azul + vermelho) e o verde (azul + amarelo). Da junção de uma cor primária com uma secundária, surgem as seis cores terciárias:

  • Amarelo-alaranjado: Amarelo + Laranja
  • Amarelo-esverdeado: Amarelo + Verde
  • Vermelho-alaranjado: Vermelho + Laranja
  • Vermelho-arroxeado: Vermelho + Roxo
  • Azul-arroxeado: Azul + Roxo
  • Azul-esverdeado: Azul + Verde

Divisões

As cores podem ser dividas em temperatura e harmonia, as cores quentes são o laranja, amarelo e vermelho, e as frias são o verde, azul e violeta.

Uma cor é complementar da outra quando são imediatamente opostas no círculo cromático. As cores análogas são aquelas imediamente vizinhas.

O site pode ter uma paleta formada por cores monocromáticas, que são cores extremamente parecidas, cores triádicas, que formam um triângulo no círculo cromático, cores análogas mais uma complementar, que é a junção de 3 cores vizinhas mais uma oposta e entre outras opções.

Onde encontrar

Há inumeros sites que podem fornecer as paletas de cores para o desenvolvimento web, porém o conhecido é o Adobe Color.

Representando uma cor

Uma cor pode ser representada pela quantidade de azul, vermelho e amarelo, com números entre 0 a 256 e também com HSL, que é matiz, saturação e luminância.

No editor de códigos é possível fazer uma cor degradê com a tag <background-image>.
