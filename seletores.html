Seletores Personalizados em CSS

Personalizando

No CSS, é possível identificar um determinado elemento com um id, ou agrupar elementos múltiplos que tenham características semelhantes com um class. Com o id, é possível criar um estilo isolado somente para o elemento identificado. Já um class, utiliza-se para identificar uma classe para um ou mais elementos, compartilhando entre eles características.

Propriedades herdadas

Como explicado anteriormente, existe o CSS externo, local e o inline, caso seja utilizado em um mesmo arquivo os três tipos, o que ficará valendo é o inline, isso também acontece com o id e o class, no caso, o id sempre terá suas configurações como as principais, depois o class e por último os do body ou as tags semâticas em geral, as demais configurações que estiver além das do id, serão acrescentadas adjuntas.

Pseudo-classes e psseudo-elementos

Uma pseudoclasse CSS é uma palavra-chave adicionada após o sinal de dois pontos nas declarações de um seletor, elas especificam um estado especial de algum elemento. Já um psseudo-elemento CSS, é uma palavra-chave adicionada após dois sinais de dois pontos, elas permitem a formatação de um pedaço específico do elemento referenciado.

Personalizando

Um exemplo: com a tag >a<, criamos uma âncora para levar a um vídeo do youtube, no CSS, escrevemos a pseudo-classe "a:visited{color: red;}", a pseudo-classe atribuiu a função de que depois que o usuário acessar o hyperlink, a âncora ficará da cor vermelha.
